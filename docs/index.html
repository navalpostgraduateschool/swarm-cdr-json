Vehicle JSON Generator (VJG)

The Vehicle JSON Generator is a simple yet powerful tool developed in MATLAB to support the creation of JSON configuration files for military ground and aerial vehicles. These configurations are used in the Swarm Commander Tactics Software for simulating tactical engagement scenarios.

What It Can Do

  • Select vehicle category and variant from dropdowns
  • Edit performance metrics like speed, weapon power, or weight
  • Save each configuration as a uniquely named JSON file
  • Reload previously created JSON files for continued editing

Why It Matters

Swarm Commander requires strict input schema. The VJG eliminates manual formatting errors and accelerates scenario preparation for instructors, analysts, and developers working in defense simulation environments.
